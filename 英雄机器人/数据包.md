发送的数据包

``   uint8_t send_bytes[] = {0xAA,        // frame head
                            0x00,        // mode
                            0x00, 0x00,  // yaw
                            0x00, 0x00,  // pitch
			    0x00,	 // speed
                            0xA5};       // frame tail``    

**发送的角度都是乘50之后的**

接收的数据包

`` /*
    0: head 0xAA
    1: ID: < 10 --> red  > 10 --> blue  &  mode  0 --> rune  1 --> auto
    2, 3 --> yaw
    4, 5 --> pitch
    6 --> velocity
    7: tail 0x55
    */``

**发送的角度都是除以50之后的**
